# Todo List 浏览器扩展

中文 | [English](README_EN.md)

一个功能强大的浏览器扩展，帮助你高效管理日常任务和计划事项。

## ✨ 功能特点

- 📌 **计划事项管理**：添加和跟踪重要的计划任务
- 🔄 **每日必做任务**：管理需要每天完成的例行任务
- 🌓 **自适应主题**：支持亮色/暗色模式，自动适应系统设置
- 🎨 **现代化界面**：简洁优雅的设计，流畅的动画效果
- ✅ **任务状态追踪**：清晰显示任务完成状态

## 🚀 安装说明

1. 下载此扩展的源代码
2. 打开浏览器的扩展管理页面
3. 启用开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择本项目的文件夹

## 💡 使用方法

1. 点击浏览器工具栏中的扩展图标打开Todo List
2. 在相应的输入框中输入任务内容
3. 点击"添加"按钮或按回车键添加任务
4. 任务添加后可以：
   - 标记为完成
   - 删除任务

## 🎯 技术特点

- 使用原生JavaScript开发，无需额外依赖
- 采用现代CSS特性，包括CSS变量和媒体查询
- 支持深色模式，提供出色的夜间使用体验
- 使用Inter和JetBrains Mono字体，提供优秀的排版效果

## 🛠️ 项目结构

```
├── manifest.json     # 扩展配置文件
├── popup.html       # 扩展弹出窗口的HTML
├── popup.js         # 弹出窗口的JavaScript逻辑
├── background.js    # 后台脚本
└── image_fx.jpg     # 资源文件
```
